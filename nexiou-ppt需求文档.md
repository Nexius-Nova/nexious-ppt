# AI PPT 自动生成 Agent 项目需求文档

## 1️⃣ 项目概述
构建一个 **个人 AI PPT 生成 Agent**，通过 AI 模型自动将用户提供的主题或文档转换为 PPT 文件，支持：
- 自动文本分析与大纲生成
- 图像生成与排版
- Skill 插件系统（扩展功能，如讲稿生成、图表生成）
- 可视化工作流，用户可操作和微调每一步
- 前端页面风格参考 **OpenClaw WebUI**

---

## 2️⃣ 功能需求

### 2.1 核心流程
1. **用户输入**
   - 输入主题、文档、或关键内容
   - 可上传文件（txt, md, pdf ,word,图片等）
2. **文本分析模块**
   - AI 自动生成 PPT 大纲（标题 + 要点）
   - 支持用户手动修改或微调
   - 可配置摘要长度、页数、语言风格
3. **图像生成模块**
   - 根据每页内容生成对应插图
   - 支持选择图像风格（写实 / 插画 / 漫画 / 扁平化）
   - 可生成多张图供选择替换
4. **PPT 排版模块**
   - 自动排版文本和图像
   - 支持模板选择（商务 / 创意 / 教育等）
   - 可拖拽调整布局
5. **Skill 扩展模块**
   - 用户可选择并添加 Skill 插件
   - 默认 Skill：
     - 讲稿生成 Skill：自动生成每页演讲稿
     - 数据图表 Skill：生成数据可视化图表
     - 设计优化 Skill：自动优化布局、字体、颜色
   - Skill 支持链式调用和参数配置
6. **预览与微调模块**
   - 文本修改
   - 图像替换或再生成
   - 布局调整
   - 实时查看 PPT 预览
7. **导出模块**
   - 导出 PPT 文件（.pptx / PDF）
   - 支持下载或保存

---

### 2.2 可视化需求
- 左侧：流程条 + Skill 插件列表
- 中间：操作面板
  - 显示当前步骤输出（大纲、图像、布局）
  - 支持微调与操作
- 右侧：参数配置面板
  - 调整文本长度、图像风格、模板、Skill 参数
- 每一步进度和输出必须实时显示
- 支持拖拽 Skill 插件和拖拽布局

---

### 2.3 技术需求

#### 前端
- Vue 3 + Pinia + Vue Router + js
- TailwindCSS 或自定义 CSS（参考 OpenClaw WebUI 风格及配色）
- 可视化工作流组件化
- 实时显示每步进度（WebSocket 或轮询 API）
- 代码要组件化，要有自己的UI库

#### 后端
- Next.js API Routes 或 Express.js
- AI 模型调用：
  - 文本生成：GPT等api
  - 图像生成：ChatGPT-Image2.0 / Stable Diffusion
- PPT 生成：
  - Node.js: pptxgenjs
  - 可选 Python 后端: Python-pptx（通过 API 调用）
- Skill 系统：
  - 插件化接口，支持注册、执行、配置
  - 可异步调用并返回结果

#### 存储

- Mysql数据库
- S3 或本地文件存储（图片缓存、PPT 文件）
  
---

### 2.4 MVP 开发步骤
1. **第1周**：前端流程可视化 + 文本分析模块
2. **第2周**：图像生成模块 + 图像可视化
3. **第3周**：PPT 排版模块 + 模板选择 + 导出功能
4. **第4周**：Skill 系统 + 内置讲稿/图表/设计优化 Skill
5. **第5周**：预览与微调模块整合 + 实时工作流展示

---

### 2.5 可扩展功能
- 自定义 Skill 开发接口，支持用户添加新功能
- 自动生成讲稿和演讲提示
- 多模板风格切换

## 3️⃣ 代码规范
代码要可拓展、组件化、要有单独UI组件管理（选择框、输入框、按钮等）、不要使用渐变色，主题严格仿照openclaw webUI主题布局和风格，实现每一个功能要检测和测试通过后才能进行下一步。全程用中文回复
